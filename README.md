# Aurum — Gold E-commerce Home Page

Responsive React + Vite + Tailwind CSS e-commerce homepage with luxury gold theme.

## Run

```bash
npm install
npm run dev
```

## Structure

```
src/
  assets/            # product & hero images
  components/site/   # reusable UI sections
    Navbar.tsx
    Hero.tsx
    Categories.tsx
    Products.tsx
    ProductCard.tsx
    OfferBanner.tsx
    Reviews.tsx
    Brands.tsx
    Footer.tsx
    SectionHeading.tsx
  App.tsx
  main.tsx
  index.css
```
